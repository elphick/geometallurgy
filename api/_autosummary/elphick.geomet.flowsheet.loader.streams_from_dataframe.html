elphick.geomet.flowsheet.loader.streams_from_dataframe

elphick.geomet.flowsheet.loader.streams_from_dataframe(df, mc_name_col=None, interval_edges=None, n_jobs=1)[source]

Objects from a DataFrame

Parameters:

  • df (DataFrame) – The DataFrame

  • mc_name_col (Optional[str]) – The column specified contains the names of objects to create. If None the DataFrame is assumed to be wide and the mc objects will be extracted from column prefixes.

  • interval_edges (Union[Iterable, int, None]) – The values of the new grid (interval edges). If an int, will up-sample by that factor, for example the value of 10 will automatically define edges that create 10 x the resolution (up-sampled). Applicable only to 1d interval indexes.

  • n_jobs – The number of parallel jobs to run. If -1, will use all available cores.

Return type:

List[Sample]

Returns:

List of Stream objects
